Conversion from string to primitive types

Wrapper classes can be used to convert from string to primitive types by using the functions in the following ways:

String to Integer type data

Syntax: 

integer variable = Integer.parseInt(String);
integer variable = Integer.valueOf(String);
For example, 
int n;String s="24";
n=Integer.parseInt(s);
Or 
n=Integer.valueOf(s);
Thus, the variable and results n in 24 (without quotes).

String to Long type data

Syntax: 

long variable = Long.parseLong(String);
long variable = Long.valueOf(String);
For example, 
long n;String s="2465783";
n=Long.parseLong(s);
or 
n=Long.valueOf(s);
Thus, the variable and results n in 24657834 (without quotes).

String to Float type data

Syntax: 

float variable = Float.parseFloat(String);
float variable = Float.valueOf(String);
For example, 
Float n;String s="24.35";
n=Float.parseFloat(s);
Or 
n=Float.valueOf(s);
Thus, the variable and results n in 24.35 (without quotes).

String to Double type data

Syntax: 

double variable = Double.parseDouble(String);
double variable = Double.valueOf(String);
For example, 
Double n;String s="24.315467398";
n=Double.parseDouble(s);
Or 
n=Double.valueOf(s);
Thus, the variable and results n in 24.315467398 (without quotes).

Note: Conversion from string to primitive type data is only possible when it contains only number (a character or a digit). If these functions if tried with alphanumeric string then it throws NullFormatException. It must be kept in mind that value of function is available in Java 1.5 version and student value of function year come period system must have Java version 1.5 or more.

Conversion from primitive type data to string

A primitive type data can also get converted into string type in order to apply string functions on the digits of the number. This process is as shown below:

Integer type data to string type

Syntax: 

String variable = Integer.toString(int);
For example, 
int n=24; String s;
s = Integer.toString(n);
Here, an integer type number (n=24), gets converted to a string. Hence, s = "24". Now, any mathematical operation with the variable 's' cannot be performed.

Long type data to string type

Syntax: 

String variable = Long.toString(long);
For example, 
long n=246935; String s;
s = Long.toString(n);
Here, an long type number (n=246935), gets converted to a string. Hence, s = "246935".

Float type data to string type

Syntax: 

String variable = Float.toString(float);
For example, 
float n=24.35; String s;
s = Float.toString(n);
Here, an float type number (n=24.35), gets converted to a string. Hence, s = "24.35".

Double type data to string type

Syntax: 

String variable = Double.toString(double);
For example, 
double n=24.3452781; String s;
s = Double.toString(n);
Here, an double type number (n=24.3452781), gets converted to a string. Hence, s = "24.3452781".

Example:

Program in Java to enter a number and display the sum of the original number and the reversed number. For example, original number = 345 then reversed number = 543. Hence, the sum = 888. 

import java.io.*;
public class sum
{
	public static void main(String args[])throws IOException
	{
		InputStreamReader read = new InputStreamReader(System.in);
		BufferedReader in = new BufferedReader(read);
		int a, m, n, i, t = 0;
		int m = 0;
		String w, p = "";
		char c;
		System.out.println ("Enter a number");
		n = Integer.parseInt(in.readLine());
		w = Integer.toString(n);
		a = w.length();
		for(i=0;i<a;i++)
		{
			c = w.charAt(i);
			p = c+p;
		}
		t = Integer.parseInt(p);
		m = n+t;
		System.out.println("The sum of the original number and the reversed number ="+m);
	}
}
